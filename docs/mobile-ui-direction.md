# Mobile UI Direction

## Visual Thesis
Calm civic operations interface with soft environmental tones, map-first orientation, and restrained service-management surfaces.

## Content Plan
- first surface: live city map and filters
- support surface: complaint detail with status, image, comments, and support count
- action surface: report issue flow with offline save
- trust surface: notification history and service updates

## Interaction Thesis
- map workspace acts as the primary canvas
- report issue CTA stays persistent for fast capture
- detail and notification panels use clean, quiet surfaces with minimal chrome
